About the Formats

Read about the different formats and their common uses.


BIN Binary

The binary numeral system uses only two symbols for all expressions, usually 0 and 1, called bits. Any number can be represented this way.

Any kind of machine capable of being in two (mutually exclusive) states can represent binary numbers. Thus, most computers and other computer-based devices use the binary system.

Sometimes, a suffix b or B is added to the representation of a binary number.

Source: Wikipedia

OCT Octal

The octal numeral system uses 8 symbols and the digits 0 to 7 to represent all numbers. It's very easy to form octal numbers from binary numbers by grouping them in triplets, starting from the right. If needed, 1-2 zeroes can be added to the left to finish a triplet.

Because each octal digit represents three binary digits, binary data can be easily abbreviated using octal numbers. This is what systems like the IBM mainframes used to cut costs and conserve space.

In modern times, the hexadecimal system is usually preferred over octal, since all modern systems use words of length 16, 32 or 64 bits.

One modern use-case of octal numbers is the file permission system in Unix, also aviation.

Source: Wikipedia

HEX Hexadecimal

The hexadecimal numeral system uses 16 symbols to represent all numbers. Besides the digits 0 to 9, letters a to f are used to represent the values 10 to 15.

Every hexadecimal digit represents four binary bits (called a nibble).

Very commonly, the prefix 0x is used to represent hexadecimal numbers, as well as U+ in Unicode and # in HTML and CSS to represent color.

IPv6 addresses can be represented in groups of four hexadecimal numbers, separated by a colon :.

Source: Wikipedia

DEC Decimal

The decimal numeral system uses 10 symbols 0 to 9 and is the most commonly used system of representing (integer) numbers.

Source: Wikipedia
